# פרטיות (§29)

ברירת המחדל היא **איסוף מינימלי**.

## מה כן נאסף
- Device ID **אקראי** (לא נגזר מחומרה, לא חושף פרטים אישיים).
- גרסת ExtSync Agent.
- מערכת הפעלה כללית (למשל "windows" + גרסה כללית).
- גרסת תוסף מותקנת.
- זמן בדיקת עדכון אחרון.
- הצלחה/כשל בעדכון + Error code.

## מה לא נאסף — לעולם
- היסטוריית גלישה.
- תוכן דפים.
- טקסט שהמשתמש מקליד.
- Cookies של Chrome.
- סיסמאות.
- קבצים אישיים.
- רשימת תוספים שאינם מנוהלים ע"י ExtSync.
- אתרים שבהם המשתמש ביקר.

## Telemetry
כל נתוני שימוש שאינם חיוניים הם **opt-in** בלבד (`AgentSettings.OptInTelemetry`,
כבוי כברירת מחדל). ה-Bridge אינו אוסף מידע מדפי המשתמש.

## Device ID
נוצר פעם אחת ב-Agent (`AgentSettings.NewDeviceId`) כ-GUID אקראי. אינו מבוסס על
חומרה ואינו ניתן לקישור חוצה-מכשירים. משתמש קצה אנונימי אינו נדרש לחשבון כדי
להתקין תוסף ציבורי (§2).

## לוגים
לוגי השרת וה-Agent לא מכילים tokens/סודות (פילטר ב-`logging.py` ובמדיניות הכתיבה ב-Agent).
