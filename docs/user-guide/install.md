# מדריך למשתמש — התקנת תוסף וניהולו

## למה צריך את ExtSync Agent?
Chrome אינו מאפשר התקנה אוטומטית ושקטה של תוסף שאינו מהחנות. ExtSync Agent מכין את
התוסף עבורך, פותח את המסך הנכון ומדריך אותך לטעינה חד-פעמית. **אחרי זה כל העדכונים
אוטומטיים**.

## התקנה ראשונה
1. **התקן את ExtSync Agent** (פעם אחת): הורד מ-/download והרץ את `ExtSyncAgentSetup.exe`.
2. **פתח את קישור ההתקנה** שקיבלת מהמפתח, ולחץ "התקנה באמצעות ExtSync".
3. **אשר** את ההתקנה בחלון ה-Agent (תראה שם התוסף, מפתח, גרסה, הרשאות, ואתרים שהתוסף ניגש אליהם).
4. ה-Agent יוריד, יאמת ויחלץ את התוסף, ואז:
   - יפתח את `chrome://extensions`,
   - יעתיק את נתיב התיקייה,
   - יפתח את התיקייה ב-Explorer.
5. **ב-Chrome**: הפעל "מצב מפתח" (פינה עליונה) → "טען פריט לא ארוז" → בחר את התיקייה שנפתחה.
6. ExtSync יזהה אוטומטית שהתוסף נטען.

## עדכונים
- מתקבלים אוטומטית דרך ה-Agent. אם התוסף תומך בכך — מתעדכן ללא הפרעה; אחרת תתבקש
  להפעיל מחדש את Chrome או לטעון מחדש את התוסף.
- **בדיקת עדכונים ידנית**: כפתור "בדיקת עדכונים" במסך הראשי.
- **השהיית עדכונים**: תפריט הפעולות של התוסף.

## פעולות נוספות (תפריט התוסף ב-Agent)
- החלפת ערוץ (Stable/Beta/Nightly) — אם המפתח מאפשר.
- **Rollback** לגרסה קודמת.
- פתיחת תיקיית התוסף / דף ההרחבות.
- **הפסקת ניהול** — נשארים הקבצים אלא אם בחרת למחוק אותם במפורש.

## הרשאות
לפני התקנה תראה בדיוק אילו הרשאות התוסף מבקש ולאילו אתרים הוא ניגש. אם עדכון מוסיף
הרשאה רגישה או גישה לכל האתרים — ה-Agent יעצור ויבקש את אישורך לפני ההתקנה.

## פרטיות
ExtSync אוסף מינימום: מזהה מכשיר אקראי, גרסה, וסטטוס עדכון. אינו אוסף היסטוריית גלישה,
תוכן דפים או סיסמאות. ראו [docs/security/privacy.md](../security/privacy.md).
